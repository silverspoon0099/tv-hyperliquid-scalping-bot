HYPERLIQUID SCALPING BOT - MVP MANIFEST
================================================================================

PROJECT OVERVIEW
----------------
High-performance, low-latency Hyperliquid perpetual futures scalping bot
Receives TradingView alerts via webhooks and flips positions atomically
Built with TypeScript/Node.js for sub-20ms execution latency

CORE ARCHITECTURE
-----------------
✓ Express.js webhook server (202 async response)
✓ Async position flipping (no sequential delays, parallel execution)
✓ Hyperliquid HTTP client with connection pooling
✓ HMAC-SHA256 webhook signature verification
✓ Input validation & rate limiting (per-symbol)
✓ Structured logging (pino)
✓ Docker containerization (mainnet/testnet switching)

ENVIRONMENT VARIABLES
---------------------
Required:
  ENV                              [testnet|mainnet]
  HYPERLIQUID_[TESTNET|MAINNET]_API_KEY
  HYPERLIQUID_[TESTNET|MAINNET]_PRIVATE_KEY
  WEBHOOK_SECRET                   (for signature verification)

Optional:
  PORT                             (default: 3000)
  HOST                             (default: 0.0.0.0)
  LOG_LEVEL                        (default: info)
  MAX_POSITION_SIZE                (default: 10)
  MIN_ORDER_SIZE                   (default: 0.01)
  MAX_LEVERAGE                     (default: 5)
  RATE_LIMIT_MS                    (default: 1000)

EXPECTED LATENCIES
------------------
Webhook Receipt to Response:   < 5ms
Signature Verification:         ~1ms
Input Validation:              ~2ms
API Call (close old):          10-20ms

Total Round-Trip:              20-50ms (typical)

Note: No time.sleep() delays. All execution is async/parallel.
