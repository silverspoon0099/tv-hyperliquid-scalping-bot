TROUBLESHOOTING
════════════════

Bot won't start:
  - Check .env.local for required variables
  - Verify API keys are correct
  - Check port 3000 is available
  - Run with: LOG_LEVEL=debug npm run dev

Webhook not received:
  - Verify firewall allows port 3000
  - Check TradingView webhook URL
  - Test manually: ./test-webhook.sh
  - Check bot logs for errors

Position flip fails:
  - Check account balance
  - Verify leverage setting
  - Check order size limits
  - Review logs with LOG_LEVEL=debug

HIGH LATENCY ISSUES:
  - Check network connectivity
  - Monitor API rate limits
  - Review Hyperliquid status page
  - Consider VPS location (near Hyperliquid)

WHAT THIS BOT DOES
═══════════════════

1. Listens for TradingView webhook alerts
2. Verifies webhook signature (HMAC-SHA256)
3. Validates alert format & trading limits
4. Fetches current position
5. Closes opposite position (if exists)
6. Opens new position in requested direction
7. Returns 202 immediately (async processing)
8. Logs all actions with timestamps

All in under 50ms typically.
